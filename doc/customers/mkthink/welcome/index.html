What AgentDash Does

AgentDash is your company's AI control plane. Think of it as the operating system for a team of AI agents that work for you 24/7. Here's how it works:

🎯

You Define the Work

Create tasks in the dashboard or text your CEO agent. Describe what needs to be done — research, writing, analysis, coding.

🤖

Agents Pick It Up

Your AI agents check for new work every 30 minutes. When a task is assigned, they pick it up automatically and start working.

📊

You See Everything

The dashboard shows real-time status: who's working, what's done, what needs your review. Full transparency.

💰

You Control Costs

Set a monthly budget. Agents auto-pause when they hit the cap. No surprise bills. Token-level cost tracking.

What to Expect in Your First Week

Day 1

Setup & First Task

Your agent team is created. You'll create your first task and watch an agent complete it. Expect ~5 minutes for the first task to be picked up.

Day 2-3

Learning the Rhythm

Create 3-5 tasks per day. See how agents handle them. Some will complete quickly (research, summaries); others take longer (analysis, multi-step work). Check the dashboard 2-3 times per day.

Day 4-5

Refining Tasks

You'll learn what works. Be specific in task descriptions — agents perform better with clear instructions. Use the CoS chat to delegate higher-level work.

Week 2+

Routine

Check the dashboard morning and evening. Create tasks as needed. Review completed work. Text your CEO agent for quick requests. The system runs itself.

How Often Should You Interact?

💡 Key Insight:

Agents are not ChatGPT. They don't respond instantly. Think of them as employees who check their inbox every 30 minutes. Here's the recommended interaction pattern:

When What to Do Time
Morning (9 AM) Check dashboard. Review overnight results. Create tasks for the day. Text CEO agent if you have requests. 5-10 min
Midday (optional) Quick dashboard check. See if any tasks need your review or approval. 2-3 min
Evening (5 PM) Review completed work. Check costs. Create tasks for tomorrow. Respond to any agent questions. 5-10 min

Total daily commitment: 15-20 minutes. The agents do the rest.

What to Watch Out For

⚠️ Budget Limits

If you're using API billing, agents will pause when they hit the monthly budget cap. Check the Costs page regularly. If agents pause unexpectedly, it's likely the budget — just raise it or wait for the next billing cycle.

⚠️ Task Quality

Agents perform best with specific, detailed instructions. "Research our competitors" is vague. "Research our top 5 competitors in the consulting space and create a 2-page summary with strengths, weaknesses, and market position" is excellent.

⚠️ Agent Availability

Agents check for work every 30 minutes. They don't work instantly. If you need something fast, mark it as "critical" priority — the agent will pick it up on the next heartbeat cycle.

⚠️ Mac Mini Must Stay On

AgentDash runs on your Mac mini. If it sleeps, restarts, or loses network, agents stop working. Ensure the Mac mini is set to never sleep (System Settings → Energy → "Prevent automatic sleeping").
