**We built an AI workforce management platform. Here's what we learned.**

Every company we talked to is using AI. But nobody's managing it.

Someone on the team is running Claude. Another's got ChatGPT. There's no oversight, no budget tracking, no kill switch.

So we built AgentDash — the management layer between your business and your AI agents.

What it does:
- Spawn AI agents from templates with approval gates
- Set budgets by department with hard stops
- Track everything from a single morning dashboard
- Kill switch that halts all agents instantly
- HubSpot CRM integration out of the box
- Task dependencies that auto-route work between agents

The key insight: governance first, automation second.

Before any agent runs, we set up departments, security policies, and spending limits. Because "move fast and break things" doesn't work when AI agents have access to your CRM.

We built AgentDash on top of Paperclip (github.com/paperclipai/paperclip), an open-source agent control plane. They built the engine. We built the cockpit — Agent Factory, CRM pipeline, security policies, budget management, and guided onboarding.

Our onboarding takes one day. Not weeks. Deploy, configure, set governance, spawn agents, connect CRM, go live. We tested the entire flow end-to-end with automated validation. 28 steps. Zero failures.

We're onboarding our first client now — a mid-size B2B SaaS company deploying agents for lead qualification, content drafting, and engineering triage.

If you're a 20-100 person company that's using AI manually and wants to do it systematically, with real oversight — let's talk.
